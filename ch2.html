Основы HTML

Элемент <p>

<p></p> - создать параграф.

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<p>Параграф текста.</p>
<p>Ещё один параграф текста.</p>
</body>
</html>

Перенос строки

Тег <br/p> - добавить отдельную строку, когда нет надобности создовать новый параграф:

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<p>Параграф текста.</p>
<p>Ещё один параграф текста.</p>
<p>Здесь <br/> перенос строки.</p>
</body>
</html>

Элементы форматирования

Элементы форматирования были созданы для отображения разных стилей текста:

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<p>Обычный текст</p>
<p><b>Текст жирным шрифтом</b></p>
<p><big>Текст крупным шрифтом</big></p>
<p><i>Текст курсивом</i></p>
<p><small>Текст мелким шрифтом</small></p>
<p><strong>Важный текст</strong></p>
<p><sub>Подстрочный текст</sub></p>
<p><sup>Надстрочный текст</sup></p>
<p><ins>Вставленный текст (подчёркнутый)</ins></p>
<p><del>Удалённый текст (зачёркнутый)</del></p>
</body>
</html>

Тег <strong> - фразовый тег. Он обрамляет важный текст.

Браузеры отображают <strong> как <b> и <em> как <i>.
Тем менее, эти теги имеют различные функции:
<b> и <i> обозначают текст жирным шрифтом и курсивом соответственно, тогда как <strong> и <em> указывают на важность текста.

Заголовки

В HTML спользуеся 6 уровней заголовков, которые организованы по уровню важности:
<h1>, <h2>, <h3>, <h4>, <h5> и <h6>.
В нижеприведённом коде использованы все 6 заголовков:

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<h1>Заголовок 1</h1>
<h2>Заголовок 2</h2>
<h3>Заголовок 3</h3>
<h4>Заголовок 4</h4>
<h5>Заголовок 5</h5>
<h6>Заголовок 6</h6>
</body>
</html>

Поисковые системы придают заголовкам большое значение: используют их для индексации структуры страницы и её контента.

Линии

Чтобы добавить вдокумент горизонтальную линию, используется тег <hr/>

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<h1>Заголовок 1</h1>
<h2>Заголовок 2</h2>
<h3>Заголовок 3</h3>
<h4>Заголовок 4</h4>
<h5>Заголовок 5</h5>
<h6>Заголовок 6</h6>
<p>Параграф текста.</p>
<hr/>
<p>Параграф текста.</p>
</body>
</html>

В HTML5, тег <hr/> обрамляет тематический разрыв.

Коментарии

Комментарии браузером не отображаются, но они помогают в составлении документа и дают возможность добавить описания, делать заметки, напоминания и прочее.

<!-- Текст Вашего коментария -->

Пример:

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<p>Параграф текста.</p>
<hr/>
<p>Ещё один параграф текста.</p>
<!-- Текст Вашего коментария --> </body>
</html>

Элементы

HTML - элементы

Это строительные блоки любого HTML-документа.
HTML - документ состоит из открывающего и закрывающего тегов, которые обромляют его содержимое.
HTML - окумент может состоять из вложенных в друг друга HTML - элементов.

<html>
<head>
<title> Первая страница </title>
</head>
<body>
<p>Параграф текста.<br/></p>
</body>
</html>

Атрибуты

HTML - атрибуты

Атрибуты содержат дополнительную информацию об элементе или теге, а также изменяют их. Большинство атрибутов имеют значения; значение изменяет атрибут.

<p align="center">
Этот текст выровнен по центру
</p>

В этом примере значение атрибута "center" указывает на то, что содержимое элемента р долэжно быть выровнено по центру.

Атрибуты всегда пишутся в открывающем теге и имеют следующую парную структуру:
name="value"

Единицы измерений в атрибутах

В качестве примера, добавим в HTML-документ горизонтальную линию шириной в 50 пикселей.
Для этого нам понадобится атрибут ширины (width)

<hr width="50px" />

Ширина элемента может быть определена в процентах:

<hr width="50%" />

Атрибут выравнивания

Арибут выравнивания указывает на то, как текст будет выровнен в документе.
В приведённом ниже примере параграф текста будет выровнен по центру, а строка - по её правому краю.

<html>
<head>
<title>Атрибуты</title>
</head>
<body>
<p aligan="center">Параграф текста.<br/>
<hr width="10%" aligan="right" />Строка текста
</p> </body>
</html>

Атрибут aligan тега <р> не поддерживается в HTML5
